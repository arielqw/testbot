You can configure some run parameters here:


- TESTS_PATH - Relative path to tests .scm files directory

- OUTPUTS_PATH - Relative path to petite valid outputs directory

- CODEGEN_PATH - Relative path to code-gen output .c files directory

- CODEGEN_EXE_PATH - Relative path to compile-scheme-file .o files directory

- CODEGEN_OUTPUT_PATH - Relative path to code-gen results of running .o files directory

- compilerDirectoryPath - Relative path to directory where compiler.scm is
